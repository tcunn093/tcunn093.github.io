Privacy Policy

Almost There! For OC Transpo does not collect, sell, or manage identifying user data.

Users of Almost There! provide the GPS coordinates of their phone, which is then compared to a database of GPS coordinates for OC Transpo bus stops. Users cannot be identified by the information provided to the server

We do not use these coordinates to track users in any capacity; they are simply sent to the server for processing. The GPS activity of users is not stored in a database.
